# analyze-cli 검증 체크리스트

## REQ-AC-001: CLI 인자 파싱

- [ ] `--days` 파싱 (기본값: 30)
- [ ] `--project` 파싱 (기본값: null)
- [ ] `--project-path` 파싱 (기본값: null)
- [ ] `process.argv.slice(2)` 사용
- [ ] `runAnalysis()`에 올바른 파라미터 전달

## REQ-AC-002: 분석 헤더 출력

- [ ] 콘솔 헤더: `=== Self-Generation AI 패턴 분석 (최근 {days}일) ===`
- [ ] 분석 시작 시 출력

## REQ-AC-003: 데이터 부족 처리

- [ ] 프롬프트 5개 미만 시 안내 메시지
- [ ] 메시지: `데이터 부족: 프롬프트 5개 이상 수집 후 다시 실행하세요.`
- [ ] exit code 0으로 종료

## REQ-AC-004: 분석 결과 포맷팅 출력

- [ ] 4개 섹션: 클러스터, 워크플로우, 에러 패턴, 제안
- [ ] 데이터 존재 시에만 섹션 출력
